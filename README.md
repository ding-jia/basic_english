# Basic English 单词卡片

一个纯静态单页网页，把经典的 **Basic English（Ogden 850 词）** 做成可浏览、可搜索、可练习的单词卡片。

无需后端、无需构建、无 CDN 依赖，**离线即可使用**。

## 项目简介

Basic English 由 C. K. Ogden 提出，用 850 个核心英语单词覆盖日常英语表达。本项目将这 850 个单词整理为单词卡片，支持分类筛选、中英文实时搜索，以及翻面抽卡的练习模式，帮助你高效记忆核心词汇。

词表构成：

- **600 个名词**（n.）
- **150 个形容词**（adj.）
- **100 个动词**（v.）

## 功能特性

- **卡片网格展示**：全部单词以卡片网格呈现，每张卡片显示单词、音标、彩色词性标签和中文释义。
- **分类筛选**：按「全部 / 名词 / 形容词 / 动词」切换，并实时显示各类数量。
- **搜索框**：输入中文或英文，实时过滤匹配的单词。
- **练习模式（翻面抽卡）**：
  - 开启后进入抽卡界面，正面只显示单词和音标，点击卡片或按空格键翻面，显示词性与中文。
  - 提供「上一张 / 下一张 / 随机」按钮，并显示当前进度。
  - 抽卡范围联动分类筛选与搜索结果。
  - 快捷键：`←` / `→` 切换卡片，`Esc` 退出练习模式。
- **响应式布局**：在手机等小屏设备上自动适配。

## 文件结构

```
basic_english/
├── data.js      — 全部 850 词的 JS 数组（window.WORDS）
├── index.html   — 页面外壳
├── style.css    — 全部样式
└── script.js    — 全部前端逻辑
```

## 数据说明

词表数据存放于 `data.js`，以 `window.WORDS` 数组形式提供。每个单词为包含四个字段的对象：

| 字段 | 说明 |
|------|------|
| `word` | 英文单词 |
| `phonetic` | 音标（IPA，英式，用 `/.../` 包裹） |
| `pos` | 词性，取值 `n.` / `adj.` / `v.` |
| `meaning` | 中文翻译，多个义项以顿号分隔 |

示例：

```js
{ word: "account", phonetic: "/əˈkaʊnt/", pos: "n.", meaning: "账户、叙述、理由" }
```

## 使用方法

### 方式一：直接打开

双击 `index.html` 即可在浏览器中打开（`file://` 协议下 `data.js` 正常加载）。

### 方式二：启动静态服务

```bash
cd basic_english
python3 -m http.server 8000
```

然后在浏览器访问 <http://localhost:8000>。

## 技术栈

- **纯 HTML / CSS / JavaScript**，无任何框架或库。
- **无后端**、**无构建步骤**、**无 CDN 依赖**，完全离线可用。
- 数据以原生 JS 数组（`window.WORDS`）加载，无需网络请求。
